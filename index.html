Cisco CMX

Connected Mobile Experience (CMX) is a service that provides real time data and analytics through location tracking. This mobile application is an adminitrator application that allows users of the CMX service check on status. Not only that, it contains a mobile testing suite allowing the user to realistically test the accuracy and precision of their location services with a mobile device.

Before this app, there was no easy way to check up on the service's status on the go making the app vital. I worked full-time on developing this application from the ground up using Swift. After my internship ended, the team continued to polish the application and tweak the UI; however, all of the underlying features have remained the same.

Dopamine

Dopamine is an API that helps apps form habits among its users, allowing for greater user-retention and a better user experience. It's currently used in educational and fitness apps among other to allow for the app's users to form better habits.

I found interest in Dopamine and took the opportunity to meet up with one of the creators (Ramsay Brown - neuroscience PhD candidate at USC) to see if I could take part in the project. After joining his team, I developed the iOS version of the Dopamine client library using Objective-C. The client library allows apps to easily utilize the Dopamine API with only a few lines of code.
